Array --->([])
It is used to collect multiple elements and store them in a single variable.

Two types of an array:
1.Homogeneous Array: It will store the same data type value.
2.Heterogenous Array:It will have different data type values.

let array = [1,2,3,4,5,6]

1.map(): It is used to iterate over an array, and it will return new array.
 syntax: array_name.map((initial_value,index,array)=>{//code})

2. filter(): It is used to filter the values based on the condition, and it will return a new array that contains the filtered values.
 syntax: array_name.filter((initial_value,index,array)=> condition)

3. find(): It is used to return a single value from the array based on the condition.
 syntax:array_name.find((initial_value)=> condition)

synatx:

let details ={
    productName : "Mobile",
    price : 25000
}

Accessing the object in two ways:

     1.Dot notation
     2.Box or Bracket notation

1.Dot Notation:

synatx: object_name.key //value

clg(details.productName) //"mobile"
clg(details.price) // 25000

2.Box Notation:

syntax: object_name["key"] //value
clg(details["productName])




Promise:

It is used to convert the sync code to async code.
It is a object.

In promise, we have three states:
          1.pending Stage
          2.Fulfilled Stage
          3.Rejected Stage

1.Pending Stage
let myPromise = new Promise((res,rej)=>{
    let lunch = true;
})


2.Fulfilled promise
let myPromise = new Promise((res,rej)=>{
    let lunch = true;
    if(lunch){
        res("Fulfilled")
    }
    else{
        rej("Rejected")
    }
})


3.Rejected promise
let myPromise = new Promise((res,rej)=>{
    let lunch = false;
    if(lunch){
        res("Fulfilled")
    }
    else{
        rej("Rejected")
    }
})
        
To handle the promise we are using the (async) and (await)

// Async and await

let data = async()=>{
    let response = await myPromise;
    console.log(response);
};
data();
console.log("end");
